# 🥷 指尖忍者 (Fingertip Ninja) - Cyberpunk Edition

基于 Python + OpenCV + MediaPipe + Pygame 开发的 AI 体感互动游戏。无需鼠标键盘，只需在摄像头前挥动双手，即可化身赛博忍者，体验“双刀流”切水果的快感！

## ✨ 核心亮点

* **🤖 顶尖 AI 视觉跟踪**：底层接入 Google MediaPipe 手部追踪大模型，精准捕捉 21 个骨骼关键点，实现零延迟指尖定位。
* **⚔️ 赛博双刀流系统**：支持双手同时操作，左手青色（Cyan）、右手品红（Magenta）霓虹剑气，爽快感翻倍。
* **🖥️ 极客风 HUD 界面**：全屏扫描线遮罩、半透明暗色面板、动态霓虹发光文字，科技感拉满。
* **🏆 专属段位评价**：内置硬核评分系统，根据命中率生成个性化“损友评价”。

## 🛠️ 如何运行源码 (给开发者)

1. 克隆本项目到本地：
   
   git clone [https://github.com/你的用户名/你的仓库名.git](https://github.com/yanlinhong/Fingertip-Ninja.git)

2. 安装环境依赖

   pip install -r requirements.txt

3. 启动系统

   python ninja.py
